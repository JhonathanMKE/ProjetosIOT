TUTORIAL 

####################

[1] Install Arduino
[2] Copy Additional Boards Manager URLs to Arduino
[3] Install ESP32 by Espressif Systems on Boards Manager [Arduino]
[4] Install Esp8266 by ESP8266 Community on Boards Manager [Arduino]

[5] Install MFRC522 by GithubCommunity
[6] Install LiquidCrystal I2C by Frank de Brabander
[7] Install PubSubClient by Nick O'Leary
[8] Replace MFRC522Extended.cpp in folder UserDocuments/Arduino/libraries/MFRC522/src/

[9] Select ESP32 Dev Module and compile code
